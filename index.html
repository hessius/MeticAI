How It Works

MeticAI runs on a small computer in your home — like a Raspberry Pi, Mac, or Windows PC — and creates a private web app just for you.

1

Install

Run a one-line installer on Linux/macOS, PowerShell on Windows, or use the macOS app. It sets everything up and starts running in the background.

2

Open the Web App

Access MeticAI from any phone, tablet, or computer on your home network. No app store needed — just use your browser.

3

Snap & Brew

Take a photo of your coffee bag, pick taste tags, and let the AI create and upload a tailored espresso profile to your machine.

4

Control & Automate

Use the Control Center to manage your machine, schedule preheats, or integrate with Home Assistant and iOS Shortcuts.

Getting Started

What You Need

  • ☑️ A Meticulous Espresso Machine (connected to your network)
  • ☑️ A server to run MeticAI (Raspberry Pi, Mac, Linux, or Windows PC)
  • ☑️ A free Google Gemini API key → Get yours here (takes 30 seconds)

Installation (5 minutes)

One-line install for Raspberry Pi, Linux, and macOS:

curl -fsSL https://raw.githubusercontent.com/hessius/MeticAI/main/scripts/install.sh | bash

The installer checks prerequisites, pulls the Docker image, and starts all services automatically.

PowerShell installer for Windows 10/11 (requires Docker Desktop):

irm https://raw.githubusercontent.com/hessius/MeticAI/main/scripts/install.ps1 | iex

Runs in PowerShell 5.1+. See the Windows guide for details.

Prefer a GUI? Download the macOS installer app — no terminal required.
